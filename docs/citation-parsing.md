# 인용·준용 규정 파싱 — 한계 및 확장 방법

## 현재 구현 위치

`core/citation_parser.py`

---

## 지원하는 패턴

| 패턴 유형 | 예시 | 지원 |
|-----------|------|------|
| 타법 인용 | `「소득세법」 제27조제1항` | ✅ |
| 같은 법/령/영/규칙 | `같은 법 제27조`, `같은 영 제15조제2항` | ✅ |
| 직접 조 인용 | `제27조`, `제27조의2`, `제27조제1항제2호` | ✅ |
| 조 범위 | `제20조부터 제25조까지` | ✅ |
| 항·호·목 범위 | `제1항부터 제3항까지`, `제2호부터 제5호까지` | ✅ |
| 조 내 단독 항·호 | `제3항`, `제2호` (조번호 없는 형태) | ✅ (부분) |

## 지원하지 않는 패턴

### 1. 전항 / 다음 항

```
전항에 따라 ...
전항의 규정에 의하여 ...
다음 항에서 정하는 ...
```

**현재 동작**: 탐지 안됨.
**추가 방법**: `citation_parser.py`에 패턴 추가.

```python
# 추가할 패턴 예시
_PREV_HANG = r"(전항|전조|다음\s*항|다음\s*조)"
PREV_HANG_RE = re.compile(_PREV_HANG)

# parse_citations() 내 추가 루프
for m in PREV_HANG_RE.finditer(text):
    if m.span() in seen:
        continue
    seen.add(m.span())
    results.append(Citation(
        raw=m.group(0),
        jo="",          # 조번호 없음 (상대 참조)
        hang="전항" if "전항" in m.group(0) else "",
        span=m.span(),
    ))
```

`Citation.jo`가 빈 문자열이면 UI에서 "상대 인용"으로 별도 표시 가능.

---

### 2. 단서 참조

```
단서에 따라 ...
단서의 규정에 의하여 ...
```

**현재 동작**: 탐지 안됨. 단서는 조 번호가 없어 링크 연결 불가.
**활용 방안**: 단서 참조가 있다는 것만 알려주는 형태로 표시.

```python
_PROVISO = r"(단서에\s*따라|단서의\s*규정에\s*의하여)"
```

---

### 3. 이 조 / 이 항 / 이 호

```
이 조에서 "법인"이란 ...
이 항에서 "과세기간"이란 ...
```

**현재 동작**: 탐지 안됨.
**특성**: 자기 참조이므로 링크 불필요. 단, 정의 조항임을 표시할 수 있음.

---

### 4. 준용 규정

```
제27조를 준용한다
제27조의 규정을 준용한다
```

**현재 동작**: `제27조` 부분은 직접 인용 패턴으로 탐지됨. "준용"이라는 관계는 별도 표시 안됨.
**개선 방안**: 매칭된 인용에 `is_junyo: bool` 필드 추가.

---

## 실제로 못 찾는 케이스가 발생하면

1. 못 찾는 원문 텍스트를 그대로 제보.
2. 어떤 구문이 탐지되어야 하는지 함께 알려주면 즉시 패턴 추가 가능.

예: `"제3항의 경우에는 제1항 및 제2항을 준용한다"` → 이 문장에서 `제1항`과 `제2항`이 탐지 안 됨.

## 역방향 인용 (`find_back_citations`) 한계

현재 `ui/stage2_crossref.py`에서 호출 시 `target_hang`을 항상 빈 문자열로 전달한다.
이 때문에 `find_back_citations`의 항 레벨 필터가 비활성화 상태.

**현재 동작**: 개정 조문의 특정 항이 아닌 조 전체를 인용하는 모든 조항을 반환.
**개선 방안**: `s1_article`의 항번호를 추출해 전달하면 더 정밀한 탐색 가능.

```python
# stage2_crossref.py 개선 예시
article = st.session_state.get("s1_article", {})
jo_str = str(article.get("조번호", ""))
jo_parts = jo_str.split("의")
jo = jo_parts[0]
jo_sub = jo_parts[1] if len(jo_parts) > 1 else ""
# 항번호 추출 (개정 조문이 특정 항인 경우)
target_hang = ""  # ← 여기에 항번호 전달 시 정밀 탐색
back = find_back_citations(law_data, jo, jo_sub, target_hang)
```
